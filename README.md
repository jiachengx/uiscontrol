### uiscontrol
Control the UIS-522b via send package
# version: 1.

Usage
------
    python uicontrol.py IP Target Control

|target|Description|
|------|:----------|
|0|means UIS|
|1|means outlet 1|
|2|means outlet 2|
|3|means outlet all|

|Control|Description|
|-------|:----------|
|0|means Off|
|1|means On|
|2|means switch(i.e from On-> Off, or from Off-> on|
|3|means reset|
